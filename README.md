## Requirements
- Python >= 2.7.10
- Selenium webdriver >= 3.6.0

```
$ pip install -r requirements.txt
```

## Driver/Browser
- PhantomJS >= 2.1.1
- ChromeDriver >= 2.33
- Firefox/geckodriver >= v0.19.0
- OperaDriver >= 2.30

## Command Arguments
```
$: python Test.py <url> <browser>
```
```
EX$: python Test.py http://google.com PhantomJS
```
